# Context

This repository is a project to display animals and their data.
Commit the open changes in Git, using a descriptive message.

# Process

1. Check what the current changes are
2. Add the open changes, separate them based on features.
3. Check the diffs. ! 'git diff'
4. Formulate the commit message using conventional git messages. <example>[feature]description</example>
5. Commit. ! 'git commit -m '

# Output Format

"Done, commited with the following message: <message>"
